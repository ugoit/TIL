# GitとGithub
Gitが大本で、リモートリポジトリとローカルリポジトリをもった分散バージョン管理システム
Githubはリモートリポジトリをクラウド上に置くことにしたgitのホスティングサービスの一種で、
類似サービスは他にもある。

Github DesktopやSourceTreeなど。SourceTreeはtree機能が使いやすいらしい。
